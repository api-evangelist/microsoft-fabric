<p>The April 2026 release of the on-premises data gateway is (version 3000.314).</p>
<h2>New Features</h2>
<h3>On-premises Data Gateway Auto-Update (Admin Triggered) | (Generally Available)</h3>
<p>With this capability, gateway administrators now have <strong>greater control and flexibility</strong> over gateway lifecycle management. Instead of relying solely on automatic background updates, admins can:</p>
<ul>
<li>Manually trigger gateway updates on demand.</li>
<li>Ensure updates are applied during approved maintenance windows.</li>
<li>Better align gateway upgrades with organizational change management and compliance requirements.</li>
<li>Reduce disruption risk for mission-critical workloads.</li>
</ul>
<p>This enhancement improves operational predictability while still ensuring gateways stay current with the latest performance, reliability, and security improvements.</p>
<p>To learn more, refer to the <a href="https://learn.microsoft.com/data-integration/gateway/service-gateway-update" rel="noreferrer noopener" target="_blank">Update an on-premises data gateway</a> documentation.</p>
<h2>Power BI Desktop compatibility</h2>
<p>This update brings the on-premises data gateway up to date with the April 2026 release of Power BI Desktop.</p>
<p><a href="https://download.microsoft.com/download/d/a/1/da1fddb8-6da8-4f50-b4d0-18019591e182/GatewayInstall-26-04.exe" rel="noreferrer noopener" target="_blank">Download on-premises data gateway (standard mode)</a></p>
<p><a href="https://download.microsoft.com/download/6/0/2/602A459E-E1A3-4FB9-B07F-FC2B60881900/On-premises%20data%20gateway%20(personal%20mode)-26-04.exe" rel="noreferrer noopener" target="_blank">Download on-premises data gateway (personal mode)</a></p>
<p>This version of the gateway will ensure that the reports that you publish to the Power BI Service and refresh via the gateway will go through the same query execution logic/run-time as in the February’s version of Power BI Desktop. View the data connectivity section of Power BI’s April 2026 feature summary for updates on relevant connectors.</p>
