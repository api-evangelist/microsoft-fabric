The June 2026 release of the on-premises data gateway is version 3000.322. This new version (3000.322) continues our focus on improving gateway manageability, operational visibility, and enterprise governance while maintaining secure and reliable connectivity between on-premises data sources and Microsoft Fabric services. Windows broker authentication update The June 2026 release introduces support for Windows Web Account Manager (WAM) authentication, helping organizations improve token security and align gateway authentication with Microsoft's modern identity platform.
